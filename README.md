# InputFloat

InputFloat 是一个 macOS 应用程序，用于在屏幕上通过浮动窗口实时显示当前输入法的状态。

![InputFloat Screenshot](images/image.png)

## 功能特点

- **实时显示**：在浮动窗口中显示当前输入法。
- **自定义**：支持自定义窗口大小、透明度和颜色方案。
- **开机自启**：支持设置开机自动启动。
- **简洁设计**：轻量级，不干扰日常工作。

## 安装与构建

从 [Releases](https://github.com/Hi-Dka/InputFloat/releases) 页面下载最新版本，或者手动构建：

1. 克隆仓库
2. 使用 Xcode 打开 `InputFloat.xcodeproj`
3. 构建并运行
